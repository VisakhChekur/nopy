    Bases: ObjectProperty

    A representation of a checkbox property of a page.

    Attributes:

    Name Type Description
    id str

    The id of the property.
    name str

    The name of the property.
    checked bool

    Whether the checkbox is checked or not.
    type PropTypes

    The type of the property which will always be PropTypes.CHECKBOX.

    Bases: ObjectProperty

    The id of the property.

    The name of the property.
    relations list[str]

    The ids of the pages this property relates to.
